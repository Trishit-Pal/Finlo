# Finlo — Android App (Planned)

This directory contains the Gradle project scaffold for the future Kotlin + Jetpack Compose Android app.

## Status: Not yet implemented

The Android app is a planned phase. The current active development is on:
- `backend/` — FastAPI backend
- `frontend/` — React + Vite + Tailwind web app
- `supabase/` — Supabase schema, migrations, and RLS policies

## Planned Stack
- Kotlin + Jetpack Compose
- Room + SQLCipher for local encrypted storage
- Hilt for dependency injection
- Retrofit for API communication
- ML Kit for on-device OCR
- WorkManager for background tasks
- BiometricPrompt for auth
